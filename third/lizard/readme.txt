Lizard is a free tool for analizing source code complexity, by Terry Yin (see its original license in license.txt).

This tool is hosted at http://www.lizard.ws/#. 

Source code was downloaded from https://pypi.org/project/lizard/#files (built distribution, version 1.17.10).

In Windows distribution this tool is provided as a binary (exe) file compiled with py2exe (http://www.py2exe.org/) and python-2.7.8.

In GNU/Linux distribution source code is distributed with ZinjaI, and a system's Python interpreter is required to run it.

You can update the tool just replacing the source code (GNU/Linux) or executable (Windows) with a newer version.
