This folder contains an uncompressed version of http://en.cppreference.com site's offline archive (dated from 04-09-2017, downloaded from http://en.cppreference.com/w/Cppreference:Archives).
Some files where addeed (with a bash script ../src_extras/make_cppreference_index.sh): en/zinjai_index (list of html files and their respective titles for searching), png conversions for all svg graphics.
