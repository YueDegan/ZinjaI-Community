# ZinjaI Community (Fork de mantenimiento activo)

Este repositorio es un fork mantenido de ZinjaI, enfocado en consolidar los cambios del proyecto original y mejorar su estabilidad, mantenibilidad y evolución futura.

## Objetivos del proyecto

Los objetivos principales de este fork son:

- Integrar todos los cambios disponibles del repositorio original (SourceForge)
- Mantener una base de código actualizada y funcional
- Garantizar compilaciones reproducibles en sistemas modernos
- Generar ejecutables actualizados y operativos
- Dar mantenimiento continuo al proyecto
- Preparar el código para futuras mejoras y extensiones

## Estrategia de actualización

Este fork sigue un modelo de sincronización híbrido:

- El repositorio original se considera la fuente principal de actualizaciones
- Los cambios se integran tras revisión y pruebas
- Las modificaciones locales se enfocan en:
  - Compatibilidad con sistemas modernos
  - Corrección de errores
  - Soporte para toolchains actuales
  - Mejoras de estabilidad

## Sistema de compilación

El proyecto utiliza un sistema de compilación basado en Make.

Proceso típico de compilación:

```bash
make clean
make
```
